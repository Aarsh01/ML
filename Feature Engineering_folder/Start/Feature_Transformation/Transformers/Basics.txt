Function Transformer / Mathimatical Transformer
 These are some functions tht help the data to transform into normal distribution 
 1. Log Transform 
 2. Reciprocal Transform 
 3. Square Root Transform
 4. Custom
 
 4. Box-Cox
 5. Yeo - Johnson Transform
 
 
In sklearn documantation ,
3 types of transformers:
	a. Function Transformer - {1,2,3,4}
	b. Power Transformer - {4,5}
	c. Quantile Transformer - No discuss in this folder
	
	
How to find if data is normal?
1. sns.displot()
2. pd.skew() = 0
3. QQ plot

trf = FunctionTransformer(func=np.log1p)

clf2 = DecisionTreeClassifier()
